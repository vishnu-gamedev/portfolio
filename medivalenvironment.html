Medieval Environment

This project showcases a small medieval-themed environment created in Unreal Engine 5 using Megascan assets. The focus was on environment layout, lighting setup, and scene composition, while maintaining performance with Nanite and Lumen.

The work demonstrates asset integration, scale consistency, and visual presentation in a real-time environment. Attention was given to balancing visual quality with performance, making the scene efficient for gameplay use.
